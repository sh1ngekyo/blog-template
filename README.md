# blog-template
This template can be used to create a blog/forum based on Asp.net core (API or MVC)
# Features:
1. Role based Auth
2. Admin panel (Create, update, delete posts for Authors, manage users, site settings, pages, etc for Admins)
3. Notifications
4. Smart pagination
